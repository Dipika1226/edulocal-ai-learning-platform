# EduLocal

EduLocal is an AI-based multilingual video learning platform designed to make education accessible for regional and rural women.

## Features

- Video transcription
- Regional language translation and dubbing
- Topic-wise timestamps
- Auto-generated notes
- Quizzes and recommendations

## Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
